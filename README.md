# interview_questions_solutions

This repository contains solutions for the interview assessment questions.
All programs are command-line based and use only Python built-in modules.

## Projects Included

### 1. Log File Analyzer
- Reads a log file line by line
- Counts log levels
- Displays top 3 error messages

### 2. Expense Tracker (CLI)
- Add and view expenses
- Stores data persistently using JSON
- Shows total and category-wise spending

### 3. Tic-Tac-Toe (Two Player)
- Console-based 2-player game
- Validates moves
- Detects win and draw conditions

### 4. Password Manager (Text-Based)
- Uses a master password
- Stores encrypted credentials in a file
- Supports add, view, and search operations
